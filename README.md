# Drone Development using crossflight & ArduPilot

## Overview
Built and tested a drone using APM 2.8, Crossflight, and DJI Naza-M Lite.

## Features
- Flight stabilization
- Sensor calibration
- ESC tuning

## Components Used
- APM 2.8 Flight Controller
- Brushless Motors
- ESCs
- LiPo Battery

## Software
- ArduPilot
- Mission Planner

## What I Learned
- PID tuning
- Flight control systems
- Sensor integration
